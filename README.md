## Exercice 1 : Classes et Héritage
Créer une classe Employe avec les attributs :
- nom (String)
- salaire (double)
La classe contient une méthode afficherInfos() qui affiche le nom et le salaire.
Ensuite, créer une classe Manager qui hérite de Employe et ajoute :
- prime (double)
Redéfinir la méthode afficherInfos() pour afficher aussi la prime et le salaire total (salaire + prime).

## Exercice 2 : Liste d’objets
Créer une classe Livre avec :
- titre (String)
- auteur (String)
- annee (int)

Dans main(), créer une liste de 5 livres.
Écrire un code qui :
- Affiche tous les livres publiés après l’année 2015.
- Compte combien de livres sont écrits par un auteur donné (par exemple "Victor Hugo").
